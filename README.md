### 1. 	Выписана пара чисел: 1,\ 1. Между единицами записывают их сумму: 1,\ 2,\ 1. Повторив эту операцию ещё раз, получим набор 1,\ 3,\ 2,\ 3,\ 1. После следующей операции будут выписаны числа 1,\ 4,\ 3,\ 5,\ 2,\ 5,\ 3,\ 4,\ 1. Какова будет сумма всех чисел после n операций? 


### **Решение первой задачи**:

Для решения этой задачи можно заметить некоторую закономерность в построении числовых последовательностей после каждой операции.

Рассмотрим последовательность чисел, полученную после выполнения каждой операции:

1-я операция: 1

2-я операция: 1, 2, 1

3-я операция: 1, 3, 2, 3, 1

4-я операция: 1, 4, 3, 5, 2, 5, 3, 4, 1

Заметим, что после каждой операции находятся все числа, полученные на предыдущих операциях, включая дополнительные числа, добавленные на текущей операции. Это означает, что каждая последующая операция расширяет предыдущую последовательность, добавляя новые числа между существующими.

Теперь рассмотрим количество чисел, полученных после каждой операции:

1-я операция: 1 число
2-я операция: 3 числа
3-я операция: 5 чисел
4-я операция: 9 чисел

Заметим, что количество чисел после каждой операции можно выразить как $2^n-1$ , где n - номер операции.

Теперь найдем сумму всех чисел после n операций. Для этого нужно просуммировать все числа, полученные после каждой операции.

Сумма чисел после n операций будет равна сумме первых n элементов последовательности чисел, полученной после каждой операции.

Мы знаем, что количество чисел после каждой операции равно $2^n-1$ . Поэтому, чтобы найти сумму всех чисел после n операций, нужно просуммировать первые n элементов последовательности $2^n-1$ .

Сумма первых n элементов последовательности 2^n-1 вычисляется по формуле суммы арифметической прогрессии: $S_n = (\frac{n}{2})(a_1 + a_n)$ , где $S_n$ - сумма первых $n$ элементов, $a_1$ - первый элемент последовательности, $a_n$ - последний элемент последовательности.

В нашем случае первый элемент $a_1$ = 1, последний элемент $a_n$ = $2^n-1$ .

Таким образом, сумма всех чисел после n операций будет равна: $S_n = (\frac{n}{2})(1 + 2^n-1)$ .

Например, если $n = 4$ , то сумма всех чисел после 4 операций будет: $S_4 = (\frac{4}{2})(1 + 2^4-1) = 2(1 + 15) = 32$ .

Общая формула для вычисления суммы всех чисел после n операций будет выглядеть следующим образом:

$$S_n = (\frac{n}{2})(1 + 2^n-1)$$



### 2. **Докажите для чисел Фибоначчи тождество $F_1^2+F_2^2+\ldots+\ F_n^2=F_nF_{n+1}$**

### **Решение второй задачи**


Воспользуемся математической индукцией.

Тождество Фибоначчи: $F_1^2 + F_2^2 + \ldots + F_n^2 = F_n \cdot F_{n+1}$.

Доказательство по индукции:

Базовый случай: Проверим выполнение тождества для $n = 1$ .
Для $n = 1$ , левая часть равна $F_1^2$, а правая часть равна $F_1 \cdot F_2 = F_1^2$. Таким образом, базовый случай верен.

Индукционное предположение: Предположим, что тождество выполняется для некоторого k, то есть $F_1^2 + F_2^2 + \ldots + F_k^2 = F_k \cdot F_{k+1}$.

Индукционный шаг: Докажем, что тождество выполняется для n = k+1, используя индукционное предположение.

Рассмотрим левую часть для $n = k+1$ :
$F_1^2 + F_2^2 + \ldots + F_k^2 + F_{k+1}^2$.

Мы можем заметить, что это равно сумме из индукционного предположения ($F_1^2 + F_2^2 + \ldots + F_k^2$) и последнего члена $F_{k+1}^2$.

Заменим сумму по индукционному предположению:
$F_1^2 + F_2^2 + \ldots + F_k^2 + F_{k+1}^2 = F_k \cdot F_{k+1} + F_{k+1}^2$.

Мы можем факторизовать правую часть:
$F_k \cdot F_{k+1} + F_{k+1}^2 = F_{k+1} \cdot (F_k + F_{k+1})$.

Так как $F_{k+1} = F_k + F_{k-1}$ (свойство чисел Фибоначчи), мы можем заменить $F_{k+1}$ в последнем выражении:
$F_{k+1} \cdot (F_k + F_{k+1}) = F_{k+1} \cdot F_{k+2}$.

Таким образом, мы получили правую часть тождества для n = k+1.

Итак, мы показали, что если тождество выполняется для некоторого k, то оно также выполняется для $n = k+1$ . Таким образом, по принципу математической индукции, тождество выполняется для всех натуральных чисел n.

### 3. 	**Сколько существует способов представить натуральное число n в виде суммы единиц и двоек? Например,\ n=3: 3=1+1+1;\ 3=1+2;\ 3=2+1 – три способа!**


### **Решение третьей задачи**

Количество способов представить натуральное число n в виде суммы единиц и двоек можно рассмотреть с помощью метода динамического программирования.

Обозначим количество способов представить число n в виде суммы единиц и двоек как $S(n)$.

Базовые случаи:

Если $n = 0$, то есть нет числа для представления, то $S(0) = 1$.
Если $n = 1$, то есть только один способ представления: $1 = 1$, поэтому $S(1) = 1$.
Рекуррентное соотношение:
Для числа $n > 1$ существуют два возможных случая:

Если в представлении суммы есть хотя бы одна двойка, то оставшуюся сумму можно представить с помощью $S(n-2)$ способов, так как мы уменьшили число на 2.
Если в представлении суммы нет двоек, то оставшуюся сумму можно представить с помощью $S(n-1)$ способов, так как мы уменьшили число на 1.
Суммируя эти два случая, получаем рекуррентное соотношение:
$S(n) = S(n-1) + S(n-2)$

Вычисление:
Можно использовать динамическое программирование для вычисления значений $S(n)$ начиная с базовых случаев.

Вот пример кода на C++, который реализует вычисление количества способов представления числа n в виде суммы единиц и двоек:

```cpp
#include <iostream>
using namespace std;

int countWays(int n) {
    if (n == 0) {
        return 1;
    } else if (n == 1) {
        return 1;
    } else {
        int prev1 = 1;
        int prev2 = 1;
        for (int i = 2; i <= n; i++) {
            int curr = prev1 + prev2;
            prev2 = prev1;
            prev1 = curr;
        }
        return prev1;
    }
}

int main() {
    int n = 3;
    int ways = countWays(n);
    cout << "Количество способов представления числа " << n << " в виде суммы единиц и двоек: " << ways << endl;
    return 0;
}
```

В данном примере мы вычисляем количество способов представления числа 3 в виде суммы единиц и двоек. Результат будет 3, что соответствует представлениям: 3=1+1+1, 3=1+2, 3=2+1.

### 4. **Пусть $G\left(t\right)=a_0+a_1t+a_2t^2+\dots$  — производящая функция для последовательности $a_0, a_1, a_2,\dots$ . Найдите производящую функцию для последовательности $a_0,a_2,{a_4,\ a}_8,\dots$**

### **Решение четвертой задачи**

Для нахождения производящей функции для последовательности $a_0, a_2, a_4, a_8, \dots$ мы можем использовать метод частичной суммы.

Предположим, что у нас есть производящая функция $G(t)$ для последовательности $a_0, a_1, a_2, \dots$. Мы можем использовать ее для вычисления производящей функции для последовательности $a_0, a_2, a_4, a_8, \dots$.

Обозначим новую производящую функцию для последовательности $a_0, a_2, a_4, a_8, \dots$ как $H(t)$. Заметим, что каждый член новой последовательности находится на позиции, кратной 2. То есть $a_2$ соответствует позиции 1, $a_4$ — позиции 2, $a_8$ — позиции 3 и так далее.

Разложим $H(t)$ в ряд, учитывая, что каждый член последовательности соответствует позиции, кратной 2:

$$H(t) = a_0 + a_2t + a_4t^2 + a_8t^3 + \dots$$

Теперь давайте преобразуем $H(t)$, чтобы выразить его через $G(t)$. Заметим, что $H(t)$ содержит только четные позиции, поэтому мы можем воспользоваться свойством частичной суммы, чтобы выразить $H(t)$ через $G(t)$:

$$H(t) =G(t^2)$$

Таким образом, производящая функция $H(t)$ для последовательности $a_0, a_2, a_4, a_8, \dots$ равна $G(t^2)$.

Итак, производящая функция для последовательности $a_0, a_2, a_4, a_8, \dots$ равна $G(t^2)$.


### 5. **Найдите явный вид для последовательности, заданной рекуррентной формулой $a_n=3a_{n-1}-2a_{n-2}, n\geq2; a_0=1, a_1=1$**

### **Решение пятой задачи**

Для нахождения явного вида последовательности, заданной рекуррентной формулой $a_n = 3a_{n-1} - 2a_{n-2}$, где $n \geq 2$ и $a_0 = 1$, $a_1 = 1$, мы можем использовать метод caracteristica (характеристического уравнения).

Предположим, что решение имеет вид $a_n = r^n$, где $r$ - некоторая константа.

Подставим предположенное решение в рекуррентную формулу:
$r^n = 3r^{n-1} - 2r^{n-2}$.

Разделим обе части уравнения на $r^{n-2}$:
$r^2 = 3r - 2$.

Приведем уравнение к квадратному виду:
$r^2 - 3r + 2 = 0$.

Решим квадратное уравнение. Факторизуя его, получим:
$(r - 1)(r - 2) = 0$.

Отсюда следует, что $r_1 = 1$ и $r_2 = 2$.

Общее решение рекуррентного уравнения имеет вид:
$a_n = c_1 \cdot r_1^n + c_2 \cdot r_2^n$.

Используя начальные условия $a_0 = 1$ и $a_1 = 1$, мы можем найти значения констант $c_1$ и $c_2$. Подставим $n = 0$ и $n = 1$ в общее решение:

$a_0 = c_1 \cdot r_1^0 + c_2 \cdot r_2^0$,
$a_1 = c_1 \cdot r_1^1 + c_2 \cdot r_2^1$.

Это приводит к системе уравнений:
$c_1 + c_2 = 1$,
$c_1 \cdot r_1 + c_2 \cdot r_2 = 1$.

Решая эту систему уравнений, мы найдем значения констант:
$c_1 = \frac{1}{r_2 - r_1}$,
$c_2 = \frac{1 - c_1 \cdot r_1}{r_2}$.

Подставим найденные значения констант в общее решение:
$a_n = \frac{1}{r_2 - r_1} \cdot r_1^n + \frac{1 - c_1 \cdot r_1}{r_2} \cdot r_2^n$.
Таким образом, явный вид последовательности, заданной рекуррентной формулой $a_n = 3a_{n-1} - 2a_{n-2}$, где $n \geq 2$ и $a_0 = 1$, $a_1 = 1$, равен:
$a_n = \frac{1}{2 - 1} \cdot 1^n + \frac{1 - \frac{1}{2 - 1} \cdot 1}{2} \cdot 2^n$.

Упрощая выражение, получаем:
$a_n = 1 + (2^n - 1) \cdot 2$.

Таким образом, явный вид последовательности равен $a_n = 2^{n+1} - 1$. 


### **6. Докажите, что граф с 2n+1 вершинами, степень каждой из которых не менее n, связен**

### **Решение для шестой задачи**

Для доказательства того, что граф с $2n+1$ вершинами, степень каждой из которых не менее $n$, связен, можно использовать аналогичный подход к предыдущему примеру.

Предположим, что в графе с $2n+1$ вершинами существуют две вершины, которые не соединены путем. Обозначим эти вершины как $A$ и $B$.

Из условия графа следует, что каждая вершина имеет степень, не менее $n$. Таким образом, каждая вершина соединена с не менее чем $n$ другими вершинами.

Рассмотрим две ситуации:

Если в пути, соединяющем вершину $A$ с вершиной $B$, есть другая вершина $C$, то граф все еще остается связным, так как вершины $A$, $B$ и $C$ образуют путь, связывающий все три вершины.


Если в пути, соединяющем вершину $A$ с вершиной $B$, нет других вершин, то каждая из вершин $A$ и $B$ должна быть соединена с не менее, чем $n$ другими вершинами, по условию степени графа. Таким образом, общее количество вершин в графе будет не менее $2n+2$, что противоречит исходному условию, что в графе $2n+1$ вершина.


**Таким образом** , в обоих случаях мы приходим к противоречию. Значит, граф с $2n+1$ вершинами, степень каждой из которых не менее $n$, является связным.

### **7. Докажите, что в графе с $n\geq2$ вершинами по крайней мере две вершины имеют одинаковую степень.**

###  **Решение для седьмой задачи**

Для доказательства этого утверждения воспользуемся **принципом Дирихле** .

Предположим, что в графе с $n$ вершинами все вершины имеют различные степени. В таком случае, каждая вершина может иметь степень от $0$ до $n-1$, так как в графе всего $n$ вершин. Но так как у нас имеется $n$ вершин и $n-1$ возможных значений для степени, по принципу Дирихле должно существовать как минимум две вершины с одинаковыми степенями.

Таким образом, мы пришли к противоречию с предположением о том, что все вершины имеют различные степени. Значит, в графе с $n\geq2$ вершинами по крайней мере две вершины обладают одинаковой степенью.


### **8. Найдите число неизоморфных остовных деревьев в графе $К_4-e$ , где e – любое ребро.**


###  **Решение для восьмой задачи**

Граф $K_4-e$ обозначает граф, полученный из полного графа $K_4$ удалением одного ребра $e$.

Чтобы найти число неизоморфных остовных деревьев в таком графе, мы можем рассмотреть все возможные способы выбора ребра $e$ для удаления и посчитать количество остовных деревьев в полученных графах.

Исходный граф $K_4$ имеет $4$ вершины и $6$ ребер. Поскольку каждое остовное дерево содержит все $4$ вершины, то число остовных деревьев в графе $K_4$ равно $4^{4-2} = 16$ по формуле Кэли.

Теперь рассмотрим каждый возможный вариант удаления ребра $e$ из графа $K_4$:

Если удалим ребро, соединяющее вершину $1$ и вершину $2$, то получим граф с двумя компонентами связности: вершины $1$ и $2$ будут отдельными компонентами, а вершины $3$ и $4$ будут образовывать вторую компоненту. В таком случае, каждая компонента связности будет иметь $2$ вершины, и количество остовных деревьев будет равно $2^{2-1} = 2$ для каждой компоненты. Общее число остовных деревьев в этом случае будет $2 \cdot 2 = 4$.

Если удалим ребро, соединяющее вершину $1$ и вершину $3$, то получим граф с одной компонентой связности, в которой присутствуют вершины $1$, $3$ и $4$. В таком случае, количество остовных деревьев будет равно $3^{3-2} = 3$.

Если удалим ребро, соединяющее вершину $1$ и вершину $4$, то получим граф с одной компонентой связности, в которой присутствуют вершины $1$, $2$ и $3$. В таком случае, количество остовных деревьев будет равно $3^{3-2} = 3$.

Таким образом, суммируя результаты для каждого варианта удаления ребра $e$, мы получаем общее число неизоморфных остовных деревьев в графе $K_4-e$: $16 + 4 + 3 + 3 = 26$.

### **9. Сколько атомов водорода в молекуле ароматического амина $C_{12}H_nNO$ , если в его в структурной формуле есть одна $С=0$ связь и одно бензольное кольцо.** 

###  **Решение для девятой задачи**

Для определения количества атомов водорода в молекуле ароматического амина $C_{12}H_nNO$, где имеется одна $C=O$ связь и одно бензольное кольцо, нам необходимо проанализировать структурную формулу и выяснить, сколько атомов водорода связано с атомами углерода.

В формуле $C_{12}H_nNO$ имеется одна аминогруппа $-NH$, которая связана с атомом углерода и имеет один атом водорода. Кроме того, есть одна $C=O$ связь, которая также содержит один атом водорода. Таким образом, суммарное количество атомов водорода, связанных с атомами углерода, равно 2.

Однако, в данном случае у нас также присутствует бензольное кольцо, состоящее из 6 атомов углерода и 6 атомов водорода. В ароматическом бензольном кольце каждый атом углерода связан с одним атомом водорода, и наоборот. Таким образом, в бензольном кольце имеется 6 атомов водорода.

Итак, суммарное количество атомов водорода в молекуле ароматического амина $C_{12}H_nNO$ с одной $C=O$ связью и одним бензольным кольцом равно 2 (связанных с атомами углерода) плюс 6 (в бензольном кольце), что составляет 8 атомов водорода.

### **10. Докажите, что на рёбрах эйлерова графа можно расставить стрелки так, чтобы выполнялись следующее условие: для каждой вершины числа входящих и выходящих рёбер равны.**


###  **Решение для десятой задачи**

Для доказательства данного утверждения о расстановке стрелок на ребрах эйлерова графа с условием, что число входящих и выходящих ребер для каждой вершины равно, мы можем воспользоваться следующим алгоритмом:

Рассмотрим произвольный эйлеров граф. В эйлеровом графе существует эйлеров цикл, который проходит через каждое ребро ровно один раз.
Выберем одну из вершин графа в качестве стартовой вершины и начнем обход по эйлерову циклу.

При прохождении каждого ребра в эйлеровом цикле установим стрелку в направлении обхода цикла.

Обратим внимание, что каждое ребро входит и выходит из каждой вершины ровно один раз. При этом, если встречается вершина, через которую уже прошли, то мы должны продолжить обход по циклу без изменения направления стрелки.

После завершения обхода по эйлерову циклу, на каждом ребре графа будет установлена стрелка в направлении обхода, и для каждой вершины количество входящих и выходящих ребер будет равно.

Таким образом, мы можем расставить стрелки на ребрах эйлерова графа таким образом, чтобы для каждой вершины количество входящих и выходящих ребер было равно.
